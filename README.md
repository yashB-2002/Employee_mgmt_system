# EMPLOYEE MANAGEMENT SYSTEM (CRUD APPLICATION)

## KEY FEATURES 
- CREATE NEW USER
- UPDATE EXISTING USER DETAILS
- REMOVE RECORDS 

## TECHNOLOGIES USED
- REACT
- REACT-BOOTSTRAP(STYLING)
- REACT-TOASTIFY(ALERTS)
